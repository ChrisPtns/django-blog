# Django Blog - Εργασία 4

Αυτό είναι το πρώτο μέρος της εργασίας για τη δημιουργία ενός Blog με το framework Django.

## Τι περιλαμβάνει
- Δημιουργία του κεντρικού project (`mysite`)
- Δημιουργία της εφαρμογής (`blog`)
- Μοντέλο `Post` στη βάση δεδομένων για την αποθήκευση των άρθρων
- Σύνδεση του μοντέλου με το περιβάλλον διαχείρισης (Admin Panel)

## Πώς να το τρέξετε τοπικά

1. Εγκαταστήστε το Django στο περιβάλλον σας:
   `pip install django`

2. Εφαρμόστε τις αλλαγές στη βάση δεδομένων:
   `python manage.py migrate`

3. Σηκώστε τον server:
   `python manage.py runserver`
